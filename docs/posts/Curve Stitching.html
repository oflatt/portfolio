Curve Stitching

Curve Stitching Animation

2017

Language: Racket

Source: https://github.com/oflatt/curve-stitching
An animation based upon any mathematical equation that stitches the curve using its tangent lines. It then draws the entire stitched curve each frame, offsetting the position of the points relative to time. Finally, using points on the tangent lines, it draws connecting lines for an added effect. It is fun to watch because of its seeming complexity arising from simple mathematics.
